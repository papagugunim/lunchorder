# 🍫 롯데제과 러시아법인 점심 주문 시스템

Happy Moments 러시아 모스크바 지사의 자동화된 점심 주문 관리 시스템입니다.

**현재 버전: v0.2**

## 주요 기능

### 기본 기능
- 📅 러시아 공휴일 및 주말 자동 감지
- 👥 직원별 주문 관리 (7명)
- 👤 손님 주문 추가 기능
- 📱 WhatsApp 개별 알림 (주문 링크 포함)
- 🌏 한글/러시아어 이중 언어 출력
- 📊 주문 통계 및 이력 관리
- ⏰ 자동 마감 시간 관리 (12:20 PM)
- 🔔 마감 20분 전 자동 알림

### v0.2 신규 기능
- 📊 **구글 스프레드시트 자동 연동** - 주문 데이터 자동 저장 및 장기 통계 분석
- 🎨 **관리자 UI 개선** - 직원 주문 현황과 WhatsApp 알림 통합
- 🔗 **WhatsApp 메시지에 주문 링크 포함** - 원클릭 주문 가능
- 📝 **식당용 메시지 간소화** - 70% 길이 감소
- 🌐 **러시아어 회사명 표기** - Лотте КФ Рус
- ⏱️ **근무일 주문 항상 가능** - 마감 시간과 무관하게 주문 가능

## 사용 방법

1. 웹페이지 접속
2. 직원 탭에서 본인 이름 선택 후 메뉴 주문
3. 관리자 탭에서 전체 주문 현황 확인
4. WhatsApp으로 식당에 주문 전송

## 기술 스택

- HTML5
- CSS3 (Lotte Confectionery 브랜딩)
- Vanilla JavaScript
- LocalStorage for data persistence
