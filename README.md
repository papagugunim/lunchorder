# 🍫 롯데제과 러시아법인 점심 주문 시스템

Happy Moments 러시아 모스크바 지사의 자동화된 점심 주문 관리 시스템입니다.

## 주요 기능

- 📅 러시아 공휴일 및 주말 자동 감지
- 👥 직원별 주문 관리 (7명)
- 👤 손님 주문 추가 기능
- 📱 WhatsApp 개별 알림
- 🌏 한글/러시아어 이중 언어 출력
- 📊 주문 통계 및 이력 관리
- ⏰ 자동 마감 시간 관리 (12:20 PM)
- 🔔 마감 20분 전 자동 알림

## 사용 방법

1. 웹페이지 접속
2. 직원 탭에서 본인 이름 선택 후 메뉴 주문
3. 관리자 탭에서 전체 주문 현황 확인
4. WhatsApp으로 식당에 주문 전송

## 기술 스택

- HTML5
- CSS3 (Lotte Confectionery 브랜딩)
- Vanilla JavaScript
- LocalStorage for data persistence
